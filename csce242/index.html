Assingments

Assignment 1: Basic HTML

This assignments goal was to have students correctly set up a github repository and clone that repository in there IDE. This assignment also required the students to be able to develop a basic website using fundamental HTML properties that we went over in class

Assignment 2: Basic CSS

This assignment requires students to create a webpage featuring an `index.html` with an associated stylesheet. The page must include a header, subheader, at least two paragraphs, two formatted images, and a table with specific dimensions. Additionally, a consistent color theme, style within the navigation, and images with borders and padding are all required by using all three CSS techniques (element, ID, class) to ensure the page is visually appealing and well-structured.

Assignment 3: Layout and Flexbox

This third assignment is meant to give students a better understanding of the way that the div's and flexboxes do. Students are tasked to replicate another website by using these techniques to display the information in the same style.

Assignment 4: Recreate CSS page

For this assignment we had to observe a UofSc webpage and recreate this page on our own. All of the elements and divs also had to react appropriately when converged onto smaller screens.

Assignment 5: Introduction to Java

For this introduction to java assignment we had to be able to demonstrate our understanding of how script.js files are able to alter what we see on the index.html page.
